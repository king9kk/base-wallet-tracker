# ❓ Frequently Asked Questions

## General

**Q: What is Base Wallet Tracker?**
A: A free, open-source tool to track your wallet activity on Base network.

**Q: Is it safe to use?**
A: Yes! We never store your private keys. All data stays in your browser.

**Q: Do I need to connect my wallet?**
A: No! You can search any public wallet address without connecting.

## Wallet

**Q: Which wallets are supported?**
A: MetaMask, Coinbase Wallet, and any EIP-1193 compatible wallet.

**Q: Why is my balance showing 0?**
A: Make sure you are on Base Mainnet (Chain ID: 8453).

**Q: Can I track multiple wallets?**
A: Yes! Use the watchlist feature to monitor multiple addresses.

## Transactions

**Q: Why are my transactions not showing?**
A: Only Base Mainnet transactions are shown. Testnet txs are not displayed.

**Q: How far back does transaction history go?**
A: All transactions from the Base genesis block are available.

**Q: Why is a transaction showing as failed?**
A: Failed transactions ran out of gas or hit a contract error.

## NFTs

**Q: Why are my NFTs not showing images?**
A: We show collection info from BaseScan. Images require IPFS metadata.

**Q: Which NFT standards are supported?**
A: ERC-721 and ERC-1155 tokens on Base Mainnet.

## Technical

**Q: Is there an API?**
A: We use the public BaseScan API. Get your key at basescan.org.

**Q: Can I self-host this?**
A: Yes! Clone the repo and open index.html in any browser.

**Q: How do I report a bug?**
A: Open an issue on GitHub with steps to reproduce.
