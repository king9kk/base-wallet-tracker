# 🚀 Deployment Guide

## Base Wallet Tracker - Deployment

### GitHub Pages (Free)
1. Go to repository **Settings**
2. Click **Pages** in sidebar
3. Source: **Deploy from branch**
4. Branch: **main** → **/ (root)**
5. Click **Save**
6. Live at: `https://king9kk.github.io/base-wallet-tracker`

### Vercel (Recommended)
1. Go to https://vercel.com
2. Click **New Project**
3. Import `base-wallet-tracker` from GitHub
4. Click **Deploy**
5. Live in ~30 seconds!

### Netlify
1. Go to https://netlify.com
2. Drag & drop project folder
3. Or connect GitHub repo
4. Auto-deploys on every push

### Environment Setup
No build step required! Pure HTML/CSS/JS.
Just open `index.html` in any browser.

### Performance Tips
- Enable gzip compression on server
- Set cache headers for static assets
- Use CDN for global distribution

### Custom Domain
1. Buy domain (Namecheap, GoDaddy)
2. Add CNAME record pointing to deployment
3. Enable HTTPS (free with Let's Encrypt)
