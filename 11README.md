# 🚀 Crypto Wallet Dashboard

A modern, feature-rich cryptocurrency wallet dashboard built with HTML, CSS, and JavaScript.

## ✨ Features

- 💼 Multi-wallet support
- 🔗 Multi-network (Ethereum, Polygon, BSC, Arbitrum)
- 📊 Portfolio analytics & allocation
- 💱 Token swap utilities
- 🖼️ NFT gallery & metadata
- 🔔 Price alerts & notifications
- ⛽ Gas tracker & estimator
- 📜 Transaction history & filtering
- 🌙 Dark/Light theme support
- 💾 Local storage persistence

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- CSS Variables for theming
- ES6 Modules
- LocalStorage API
