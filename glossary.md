# 📖 Web3 Glossary

A reference guide for common Web3 and Base network terms.

## A
**Address** - A unique 42-character identifier starting with 0x for a wallet or contract.
**ABI** - Application Binary Interface; defines how to interact with a smart contract.
**Airdrop** - Free token distribution to wallet addresses.

## B
**Base** - An Ethereum Layer 2 network built by Coinbase.
**Block** - A batch of transactions recorded on the blockchain.
**Bridge** - A tool to transfer assets between different blockchains.

## C
**Chain ID** - A unique number identifying a blockchain network. Base Mainnet = 8453.
**Contract** - Self-executing code deployed on the blockchain.

## D
**DeFi** - Decentralized Finance; financial services without intermediaries.
**DEX** - Decentralized Exchange for trading tokens.

## E
**ENS** - Ethereum Name Service; human-readable wallet names like `vitalik.eth`.
**ERC-20** - Standard for fungible tokens on Ethereum/Base.
**ERC-721** - Standard for NFTs on Ethereum/Base.

## G
**Gas** - Fee paid to process transactions on the blockchain.
**Gwei** - Unit of ETH. 1 ETH = 1,000,000,000 Gwei.

## L
**Layer 2** - A scaling solution built on top of Ethereum. Base is an L2.
**Liquidity** - Available assets in a DeFi protocol for trading.

## M
**MetaMask** - Popular browser wallet for Web3.
**Minting** - Creating a new NFT on the blockchain.

## N
**NFT** - Non-Fungible Token; unique digital asset on the blockchain.
**Node** - A computer that validates blockchain transactions.

## S
**Smart Contract** - Self-executing code on the blockchain.
**Staking** - Locking tokens to earn rewards.

## T
**Transaction** - An action recorded on the blockchain.
**TVL** - Total Value Locked in a DeFi protocol.

## W
**Wallet** - Software to store and manage crypto assets.
**Wei** - Smallest unit of ETH. 1 ETH = 10^18 Wei.
